# 🎯 ATS Master: Resume Optimizer

This is an AI-powered tool designed to analyze your **Resume** against a **Job Description (JD)** to improve your matching score.

## 🚀 Key Features
- **Word Count Monitor:** Ensures your resume length is within the ideal 500-700 word range.
- **Skill Gap Analysis:** Tracks missing skills based on the provided JD.
- **Keyword Frequency:** Compares keyword frequency between your resume and the JD.

## 🛠️ How to use
1. **Paste Job Description:** Enter the job requirements in the JD text area.
2. **Upload Resume:** Upload your resume in PDF format.
3. **Analyze:** Click "Run SkillSync Report" to get your optimization plan.

## 💻 Tech Stack
- Python
- Streamlit
- PyPDF2
- Spacy
